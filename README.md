# Sorting-Cluttered-Folder-

Most of our downloads folders are clunked and messy. This code sorts the Downloads folder by their file type and sorts them into their respective folders.
This code is designed to declutter messy directories. The path can be changed in the code so that any messy folder can be organized.

FEATURE:

This code sorts the files according to their file type and stores them into dedicated file type folder which are created by the program if they are not present.

REPORT:

A final report is generated after the program execution is completed. It displays all the files moved, the no of files moved, files deleted and space cleared if so.

TECHNOLOGIES USED:

Lang: Python is used for this program.
Modules: 'os' (File System Navigation)
             'shutil' (File Moving Operation)
Note: This modules are included in the standard python library and no external module is needed to be installed              for the working of this program.
